Future reduceAsync(Iterable iterable, initialValue, combine(previous, element)) #

Reduces a collection to a single value by iteratively combining elements of the collection using the provided combine function. Similar to Iterable.reduce, except that combine is an async function that returns a Future.

Future reduceAsync(Iterable iterable, initialValue, combine(previous, element))
   => _reduceAsync(iterable.iterator, initialValue, combine);

Future doWhileAsync(Iterable iterable, Future<bool> action(i)) #

Calls action for each item in iterable in turn, waiting for the Future returned by action to complete.

If the Future completes to true, iteration continues.

The Future returned completes to true if the entire iterable was processed, otherwise false.

Future doWhileAsync(Iterable iterable, Future<bool> action(i)) =>
   _doWhileAsync(iterable.iterator, action);
